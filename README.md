Import an .ics file for a one-time event import 
Open Google Calendar on a computer. 
Click the Settings gear icon in the top right corner and select Settings. 
In the left-hand menu, click Import & export. 
Under the "Import" section, click Select file from your computer and choose the .ics file you want to import. 
Select the calendar you want to add the events to from the drop-down menu. 
Click Import.
